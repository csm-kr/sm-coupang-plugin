- [아키텍처 결정 기록](../ADR.md)

- [레거시 전체 워크플로 프롬프트](쿠팡_상세페이지_전체워크플로_스킬제작_프롬프트.md)

신규 구현에서 이 문서의 본문을 작업 지침으로 사용하지 않는다. 실제 스킬 동작은 플러그인 내부 `SKILL.md`, `references/`, `scripts/`와 위 기준 문서를 따른다.
