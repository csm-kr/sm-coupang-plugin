# 게시 전 Acceptance Matrix

| 영역 | 자동 판정 | 육안 판정 | 실패 롤백 |
|---|---|---|---|
| 승인 신선도 | 계획·QA SHA-256 일치 | 승인 범위 확인 | 해당 계획 승인 |
| 제품 동일성 | source lineage·asset ID | 실루엣·타공·봉제·옵션 | 제품 사실·촬영 |
| 주장-이미지 | claim/evidence/asset ID 100% 연결 | 관련성 80/100 이상 | 스토리보드 |
| 크롭 | 피사체 95%, 핵심 부위 100% | 의미 있는 부위 잘림 없음 | CSS 또는 자산 |
| 순서 | DOM·승인 순서 100% | 구매 질문 흐름 유지 | HTML 조립 |
| 타이포 | 360·800 오류·경고 0 | 대비·리듬·여백 | 카피·CSS |
| 모션 | 길이·대체 자산·제어 | 제품 형태 연속성 | GIF·영상 |
| 광고 | 허용 상태만 사용 | 전체 문맥 오인 없음 | 제품/콘텐츠기획 |

육안 점수는 근거가 보이지 않으면 높게 줄 수 없다. 분위기 일치나 아름다움은 주장-이미지 관련성을 대신하지 않는다.
