# 상업형 Font Lock

- 히어로 후킹: Noto Sans KR Black, 72~92px, 2~3줄, 행간 0.96~1.05
- 섹션 제목: Noto Sans KR Black, 58~76px, 행간 0.98~1.08
- 본문: Noto Sans KR Regular 또는 Medium, 25~32px, 행간 1.28~1.42
- 캡션·칩: Noto Sans KR Bold, 18~24px
- 숫자·단계: Noto Sans KR Black 또는 Arial Bold, 40~64px
- 폰트 패밀리: 한글 1개, 선택적 영문/숫자 1개
- 굵기: Regular·Bold·Black 세 단계 이내
- 강조: 제목 안의 `emphasis` 한 구절만 고채도 컬러, 중복 렌더링 금지
- 대비: 흰 면은 딥 네이비·차콜, 컬러 면은 흰색을 우선
- 정렬: 히어로·요약은 중앙축, 비교·단계·정보는 좌측축 또는 카드축 우선
- 배포 전 폰트 상업 이용 조건을 확인한다.
