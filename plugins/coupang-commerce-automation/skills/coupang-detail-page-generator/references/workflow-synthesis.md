# 정반합 기반 폐쇄형 워크플로

## 목적

각 단계의 욕구와 위험을 함께 보존해 더 강한 실행 규칙으로 합친다. `정`과 `반`을 평균내지 말고, 충돌의 원인을 증거·디자인·자산 방식으로 해결한다.

## 공통 기록 형식

| 필드 | 질문 | 완료 기준 |
|---|---|---|
| `thesis` | 반드시 지켜야 하는 원본 사실·사용자 목표·기존 브랜드는 무엇인가 | 출처 또는 사용자 확인이 있음 |
| `antithesis` | 고객 불안·경쟁 약점·정보 부족·왜곡 위험은 무엇인가 | 관찰 근거 또는 구체적 실패 조건이 있음 |
| `synthesis` | 두 조건을 함께 만족시키는 실행 결정은 무엇인가 | 한 문장으로 장면·카피·자산 방식을 지배함 |
| `proof` | 무엇을 보면 결정이 맞다고 판정할 수 있는가 | 사진·실측·출처·검증 항목과 연결됨 |
| `rollback` | 실패하면 어느 단계의 무엇만 고칠 것인가 | 직전 안전 상태와 수정 범위가 특정됨 |

## 단계별 합성 규칙

| 단계 | 정 | 반 | 합 |
|---|---|---|---|
| 제품 | 원본 정체성 | 부족한 각도·정보 | 확인 사실만 잠그고 누락은 촬영·공급사 확인으로 분리 |
| 시장 | 경쟁사 반복 주장 | 복제·과장 위험 | 구매 판단 기준만 추출하고 우리 제품 증거로 재검증 |
| 브랜드 | 카테고리 친숙성 | 브랜드 차별성 | 익숙한 정보 구조 위에 고유 약속·목소리·기억 장치 반복 |
| 카피 | 강한 전환 문장 | 규제·근거 한계 | 사진이 직접 보여 줄 수 있는 범위 안에서 구체적으로 씀 |
| 비주얼 | 새롭고 풍부한 장면 | 제품 왜곡 | 제품은 보존·합성하고 배경·보조 상황만 생성 |
| 기능 | 보기 좋은 라이프스타일 | 효용이 안 보임 | 접촉·행동·반응이 한 프레임에서 읽히는 증거 장면 사용 |
| UI | 페이지별 최적화 | 캠페인 일관성 | 브랜드 토큰은 고정하고 레이아웃 모듈만 역할별 변주 |
| 한글 | 이미지와 통합된 광고 디자인 | 오탈자·정렬 위험 | Base → Design Edit → 영역 부분 편집 최대 3회, 로컬 폴백 금지, 실패 시 BLOCKED_TEXT |
| 속도 | 한 번에 10장 제작 | 오류 전파 | 내부 게이트와 제품 보존 시험을 유지하고 실패 페이지만 회귀 |
| 완료 | 예쁜 10장 | 실제 운영에 필요한 증거 부족 | 실사진·GIF·QA까지 다음 실행이 가능한 상태로 닫음 |

## 게이트 판정

각 게이트는 `PASS`, `FIX`, `BLOCKED` 중 하나다.

- `PASS`: 필수 산출물, 추적 가능한 근거, 다음 단계 입력이 모두 존재한다.
- `FIX`: 안전한 방식으로 같은 턴 안에서 보완 가능하다. 필요한 부분만 수정한다.
- `BLOCKED`: 핵심 옵션·제품 구조·법정 고지처럼 사용자 또는 공급사 확인 없이는 고객 오인이 생긴다. 일반 정보 누락은 해당 주장·장면만 멈추고 나머지 안전 작업을 진행한다. 다만 실제 SKU 동일성이 불확실하면 제품 생성·합성 전체를 멈추고 촬영 요청 게이트로 돌아간다.

## 사용자 논의 지점

페이지별 미세 승인 요청은 하지 않는다. 다음처럼 결과 전체를 바꾸는 선택만 묶어서 논의한다.

1. 기존 브랜드 유지 또는 새 브랜드 초안
2. 타겟·대표 문제·핵심 소구 3개
3. 브랜드 이름·로고의 최종 사용
4. 10장 고정 또는 정보량에 따른 장수 확대
5. 실사진 촬영 예산과 GIF 우선순위

사용자가 일괄 진행을 요청하면 1~2는 합리적인 `working_draft`로 만들고 보고서에 결정 대기 항목을 남긴다.

## 회귀 규칙

- 사실 오류 → 제품 사실 원장으로 회귀
- 소구 추적 실패 → 시장조사·소구 맵으로 회귀
- 브랜드가 카테고리와 어긋남 → 브랜드 브리프로 회귀
- 제품 외형 오류 → 자산 전략·제품 보존 시험으로 회귀
- 실제 SKU 동일성 불확실·장면별 구조 변화 → 생성 중단, `raw_capture` 촬영 요청으로 회귀
- 기능이 안 보임 → 페이지 증거 설계로 회귀
- 한글·정렬 오류 → 해당 TY 영역만 국소 수리
- 흐름·반복 오류 → 페이지 계획과 UI 변주 규칙으로 회귀
- 실사진으로만 증명 가능한 장면 → 생성 재시도 대신 촬영 목록으로 전환
