## 1. 콘텐츠 목표와 경계

- 핵심 타깃: 골프·자전거·러닝·등산·산책 등 햇빛 아래 야외활동에서 얼굴과 목을 함께 가리면서 답답함을 줄이고 싶은 사용자
- 대표 구매 불안: 얼굴과 목을 충분히 가리면 숨쉬기 답답하지 않을까
- 핵심 소구 3개: `UPF50+`·`입/코 주변 호흡부 타공`·`총 길이 25cm 롱커버`
- 구매 저항 해소: 정확한 A~E 치수·4색 옵션·재질·관리법·시험 이미지
- 현재 범위: 콘텐츠 전략·카피·순서·자산·촬영·QA 명세
- 제외 범위: 제품 이미지 생성, GIF/영상 제작, HTML 구현, 쿠팡 업로드, 광고 집행

### 정반합 결정

| 필드 | 결정 |
|---|---|
| `thesis` | 사용자가 승인한 UPF50+·호흡부 타공·25cm 롱커버·4색과 실제 HDB-1 형태를 지킨다. |
| `antithesis` | 경쟁상품에는 호흡 불편·착용 압박 불만이 있고 HDB-1의 편안함·물빠짐·내구는 실제 SKU 간이 QA 전이다. |

## 4. 이미지 + HTML 하이브리드 원칙

워크플로 `5.3`을 적용한다. 제품 사진·시험지·GIF·영상은 외부 자산으로 두고, 제목·본문·배지·치수·색상명·사양표·주의·출처는 네이티브 HTML로 둔다.

| 요소 | 구현 | 수정 방식 |
|---|---|---|
| 실제 제품 사진·타공 매크로·4색 컷 | 외부 이미지 | 실패 자산만 교체 |
| Intertek 시험지 | 공급처 원본 이미지, 변형·AI 재생성 금지 | 원본 파일 교체만 허용 |
| 제목·본문·UPF50+ 한정 문구 | HTML 텍스트 | 이미지 재생성 없이 수정 |
| A~E 치수·사양표·관리법 | HTML 표·리스트 | 값·문구만 수정 |
| 옵션 색상 | 실제 4색 이미지 + HTML 옵션명 | 품절 시 해당 옵션 모듈 숨김 |
| GIF·영상 | 외부 자산 + 정적 대체 이미지 | 자산 단위 교체 |
| 쿠팡 정적판 | 승인 HTML을 800×2400px 10장으로 렌더링 | HTML 수정 후 해당 장만 재렌더링 |

예외적으로 시험지 자체의 인쇄 문자는 증거 원본이므로 이미지 안에 유지한다. AI 생성 이미지에는 글자·로고·인증 마크를 만들지 않는다.

## 7. 소재별 QA

모든 이미지·GIF·영상은 제작 직후 자동 QA와 육안 QA를 모두 통과해야 한다.

### 자동 QA

- 안정적 자산 ID·파일 해시·형식·크기·색상 프로필
- 원본 URL·출처·제품기획 주장 ID 연결
- GIF/영상 길이·첫 프레임 대체 이미지·HTML 참조 존재
- 대체 텍스트와 모듈 ID 존재
- Intertek 시험지 원본 해시·보고서 번호·출처 문구 연결

### 육안 QA

- HDB-1 실루엣·타공·곡선·귀 리본·롱커버·4색 동일성
- 실제 사람 픽셀·임의 로고·추가 구멍·변형된 봉제 0건
- 시험지 수치·보고서 번호·출처의 가독성과 무변형
- 주장과 증거의 1:1 연결, 냉감·KC·남녀공용·의료 표현 없음
- 상업적 크롭·조명·제품 크기·모바일 가독성

실패 시 전체 세트를 다시 만들지 않고 실패 자산만 제품 사실·촬영·자산 단계로 롤백한다.

## 8. 이미지+HTML 조립 후 통합 QA

### 자동 QA

- 승인된 콘텐츠기획 버전과 HTML·CSS·자산 manifest 해시 일치
- 끊어진 자산 링크·누락 모듈·중복 ID·대체 텍스트 누락 0건
- 360px 모바일부터 데스크톱까지 오버플로·잘림 없음
- 모션 미지원·로딩 실패 시 정적 대체 이미지 표시
- 쿠팡용 800×2400px 10장 렌더 순서·크기·파일명 일치

### 육안 QA

- `문제/기준 → 핵심 해결 → UPF 증거 → 타공 → 착용 → 롱커버 → 색상 → 정보 → 요약` 흐름
- 네이티브 HTML 카피와 이미지 증거가 모순되지 않음
- 4색·치수·재질·출처·주의가 모바일에서 읽힘
- 승인 제품명 `숨트임`은 사용할 수 있으나 미승인 브랜드명·로고는 미노출
- 과장 광고·허위 인증·후기 전재·가격 오인 없음
- 소재 QA와 통합 QA가 모두 `PASS`여야 게시 패키지로 승격

## 9. 제작 시작 게이트

현재 판정은 `CONTENT_PLAN_APPROVED_AWAITING_ASSET_EVIDENCE`다.

사용자는 다음 콘텐츠기획 항목을 2026-07-16 승인했다.

1. 핵심 타깃과 대표 불안
2. 10개 모듈의 순서·카피·증거
3. 이미지+HTML 분리와 촬영·GIF 우선순위
4. 소재별 QA와 조립 후 통합 QA
5. 승인 제품명 `숨트임`과 별도 브랜드 승인 게이트

다음 단계는 실제 SKU 자산·간이 QA 준비다. 제품 단독 정·후·측면, 타공 매크로, A~E 실측, 4색 동일 구도, 포장·케어라벨을 확보하고 주장별 게시 조건을 확인한다. 제품명 `숨트임`은 `approved`이며 이미지·HTML에 사용할 수 있다. 제안 브랜드명 `에어베일`은 계속 `proposed`이며 별도 승인 전 제품명과 결합하거나 이미지·HTML에 노출하지 않는다. 샘플 발주와 게시도 별도 사용자 승인 없이는 수행하지 않는다.
