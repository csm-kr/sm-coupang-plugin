# 상세페이지 아트디렉션 벤치마크

기본 크롭은 `assets/style-reference/style-*.png` 여섯 개다. 기준 원본은 640×24,580px의 긴 모바일 상세페이지다. 이 레퍼런스는 밝은 룩북형 사진 리듬의 기준으로 사용한다. 더 강한 전환 서사와 상업형 타이포그래피는 [commercial-ad-benchmark.md](commercial-ad-benchmark.md)와 `assets/commercial-reference/commercial-*.png`를 함께 본다. 얼굴·카피·브랜드·상품 사실이 아니라 화면을 설계하는 원리만 사용한다.

## 핵심 인상

- 밝은 창가 자연광, 부드러운 그림자, 높은 화이트 비중
- 딥 네이비 제목, 파우더 블루 강조어와 배지, 아이보리·연한 식물색 배경
- 큰 인물 사진과 큰 제품 사진을 번갈아 사용한 연속형 캠페인
- 한 섹션 안에서도 제목 → 사진 → 아이콘·카드 → 다음 사진으로 자연스럽게 연결
- 제품 전체, 착용 실루엣, 손 디테일, 원단 매크로, 라벨, 생활 장면을 충분히 크게 제시

## 레퍼런스 수준으로 올리는 우선 개선점

1. 빈 공간을 줄이되 제목 안전 여백은 유지한다. 화면 하단까지 사진·카드·다음 장면이 이어져야 한다.
2. 같은 모델 포즈를 반복하지 않는다. 히어로, 착용 확인, 손 매크로, 실내, 야외, 클로징을 다른 거리와 구도로 만든다.
3. 제품 증명 밀도를 높인다. 전체형 하나만 쓰지 말고 전체 실루엣·핵심부·원단·봉제·라벨을 분리한다.
4. 타이포 위계를 잠근다. 큰 네이비 제목, 파란 강조어, 짧은 회색 설명, 작은 영문 배지 이상으로 늘리지 않는다.
5. 레이아웃 리듬을 바꾼다. 풀 프레임, 2열 카드, 전후형, 원형 확대, 3아이콘 행, 4분할 모자이크, 정보표를 교차한다.

이 다섯 항목만으로는 상업성이 충분하지 않다. 최종 기획에는 반드시 `훅 → 문제/선택 기준 → 해결 → 증거 → 사용/저항 해소 → 구매정보 → 요약` 흐름과 `상업성 8/10` 합격선을 추가한다.

## 역할별 관찰

| 역할 | 관찰할 원리 | 현재 상품에 적용 | 복제 금지 |
|---|---|---|---|
| hero | 밝은 창가 인물, 큰 질문형 제목, 첫 화면에서 제품 노출 | 텍스트 기반 합성 모델과 현재 제품 | 얼굴, 문구, 상품 |
| problem | 2열 카드와 전환 화살표, 고민→형태 설명 흐름 | 근거 없는 문제·효과 없이 사용 맥락만 | 기능 주장, 비교 카피 |
| product | 사람 없는 플랫레이, 넓은 제품 크기, 구성 아이콘 | 현재 제품의 실제 수량과 전체형 | 예시 구성, 라벨 |
| feature | 원형 확대, POINT 라벨, 손·원단 매크로 | 확인 가능한 핵심 구조와 봉제 | 예시 소재·효과 |
| lifestyle | 실내·운전·산책·여행·카페를 짧은 카드로 배열 | 현재 상품에 맞는 비주장성 장면 4개 | 추천 대상, 적합성 주장 |
| closing | 제품 플랫레이와 인물 클로징, 3개 요약 카드 | 합성 모델과 검증된 핵심 사실 | CTA 문구, 로고 |

## 사람 정책

레퍼런스 인물은 구도에서 차지하는 비율과 조명만 관찰한다. 얼굴, 헤어, 몸, 포즈를 이미지 입력의 정체성 기준으로 사용하지 않는다. 모든 캠페인 인물은 PG-01에서 텍스트로 새로 만든 합성 모델이다.
