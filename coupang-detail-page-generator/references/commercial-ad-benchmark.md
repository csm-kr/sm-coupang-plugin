# 전환형 상업 광고 벤치마크

기준 크롭은 `assets/commercial-reference/commercial-*.png` 여섯 개다. 원본은 `상세페이지-신발.png`이며, 상품·문구·브랜드를 복제하지 않고 광고 구성과 타이포그래피 원리만 사용한다.

레퍼런스의 컬러와 장식을 그대로 브랜드로 만들지 않는다. 레퍼런스는 광고 문법의 반례·근거이며, 최종 팔레트·사진·컴포넌트는 `brand-system.json`과 `ui-guide.md`에서 현재 브랜드의 약속과 증거 태도로 다시 합성한다.

## 신발 예시에서 가져올 광고 문법

- 첫 화면은 질문·상황·상품을 동시에 보여 주고, 핵심어 하나를 고채도 강조색으로 크게 잡는다.
- 문제 장면과 선택 기준은 `X / CHECK`, 2열·2×2 카드처럼 한눈에 비교되는 모듈로 압축한다. 근거 없는 전후 효과 비교는 만들지 않는다.
- 해결 뒤에는 구조, 소재 표면, 유연성, 보관, 착용 단계처럼 서로 다른 증거를 연속 배치한다. 같은 제품 사진의 단순 크롭 반복은 증거로 세지 않는다.
- 모델 얼굴과 분위기 컷을 반복하지 않는다. 같은 모델의 동일 보행·전신 구도는 한 번만 사용하고, 이후에는 물 접촉·굽힘·결합·보관·착용 손동작처럼 제품 반응이 보이는 기능 사진으로 전환한다.
- `01·02·03`, `POINT`, 짧은 칩, 원형 확대, 화살표를 사용해 보는 순서를 안내한다. 장식보다 제품 증거가 먼저다.
- 마지막에는 색상·구성·사이즈·사용법·주의사항처럼 구매 직전 정보를 표와 카드로 정리하고 핵심 혜택을 다시 회수한다.
- 화이트 정보 구간과 고채도 컬러 구간을 교차해 긴 페이지에 박자를 만든다. 팔레트는 상품과 사용자 레퍼런스에 맞춰 바꾸되 명도 대비는 유지한다.

## 전환 서사 10단계

1. `hook`: 질문·상황·핵심 상품을 1초 안에 인지시킨다.
2. `problem_or_criterion`: 고객 고민 또는 선택 기준을 사실 범위 안에서 시각화한다.
3. `solution_overview`: 제품 전체와 가장 강한 해결 구조를 연결한다.
4. `proof_1`: 형태·결합·커버 범위 등 첫 번째 증거를 크게 보여 준다.
5. `proof_2`: 개구부·조절부·핵심 디테일을 매크로로 증명한다.
6. `proof_3`: 소재 표면·봉제·라벨·구성처럼 다른 종류의 증거를 제시한다.
7. `use_or_demonstration`: 사용 순서, 착용법 또는 실제 사용 맥락을 단계화한다.
8. `friction_reducer`: 보관·관리·휴대·옵션 중 검증된 정보를 사용해 구매 저항을 줄인다.
9. `purchase_information`: 구성·색상·사이즈·고지 정보를 표와 카드로 정리한다.
10. `recap`: 제품 전체와 검증된 핵심 3가지를 다시 보여 준다.

상품에 해당 정보가 없으면 단계를 억지로 채우지 말고, 다른 검증 사실이나 비주장성 사용 맥락으로 역할을 대체한다.

## 상업형 타이포그래피 잠금

- 워크플로 5.2의 800px 폭 기준 히어로 후킹은 74~94px, 섹션 제목은 60~78px, 본문은 26~34px, 캡션·칩은 19~25px를 목표로 한다.
- 한글 제목은 ExtraBold/Black 계열, 행간 0.96~1.08, 2~3줄 이내로 사용한다. 얇은 고딕과 넓은 행간을 메인 제목에 쓰지 않는다.
- 제목 안에서 구매 이유를 담은 정확한 구절 하나만 강조색으로 바꾼다. 강조 구절을 별도 문장으로 중복 렌더링하지 않는다.
- 한 페이지의 표시용 폰트는 한 패밀리, 굵기는 Regular·Bold·Black 세 단계 이내로 제한한다. 영문 배지는 보조 역할만 한다.
- 중앙정렬만 열 장 반복하지 않는다. 히어로·요약은 중앙축, 비교·단계·정보는 좌측축 또는 카드축을 사용한다.
- 흰 배경에는 딥 네이비·차콜, 컬러 면에는 흰색처럼 충분한 대비를 확보한다. 파스텔 위 연회색 본문은 금지한다.
- 숫자·단계 표시는 42~66px의 굵은 숫자 또는 원형 번호로 만들고, 카드 제목·본문의 시작선과 내부 여백을 반복한다.

## 화면 밀도와 모듈

- 한 패널에서 `후킹 15~22% → 증거 비주얼 55~70% → 보조 설명 10~20%`의 우선순위를 기본으로 한다.
- 페이지마다 제품 또는 사용 증거를 하나 이상 크게 둔다. 정보 카드만 있고 제품이 작게 떠 있는 페이지는 실패다.
- 열 장 중 최소 5장은 기능 증거가 화면의 주인공이어야 하며, 접촉·구조·변형·사용·휴대·실측 중 최소 4종을 사용한다.
- 비교 카드, 원형 확대, 단계 스트립, 스펙 표, 체크 요약을 교차하고 같은 모듈을 연속 사용하지 않는다.
- 의도 없는 빈 하단, 작은 사진 여러 개의 균등 나열, 제목과 본문 크기 차이가 약한 조판을 금지한다.

## 상업성 합격선

다섯 항목을 각각 `0·1·2`점으로 판정한다. `8/10` 이상이며 0점 항목이 없어야 통과한다.

1. 첫 화면 훅과 상품 인지
2. 페이지별 구매 이유의 차별성
3. 주장과 시각 증거의 1:1 연결
4. 제목·강조어·본문·라벨의 위계와 대비
5. 문제/해결부터 구매정보/요약까지의 전환 흐름

상업성 점수와 별도로 브랜드 점수 `포지셔닝 / 약속-증거 / 목소리 / 시각 시스템 / 기억 장치`를 각각 0~2점으로 평가한다. 두 점수 모두 8/10 이상이고 0점 항목이 없어야 한다.
