# 상세페이지 실행 작업 영역

이 디렉터리는 실행 프로젝트와 생성 결과를 보관한다. 원본 스킬 코드는 `../coupang-detail-page-generator/`에 있다.

- 기존 `projects/project*`는 과거 실행 기록이므로 일괄 수정·삭제하지 않는다.
- 신규 작업은 승인된 `commerce-project` ID를 사용하고 프로젝트별 디렉터리를 분리한다.
- 제품 사실 원장과 source lineage를 먼저 만들고 생성 자산을 연결한다.
- 생성 결과는 원본 사실이나 테스트 픽스처로 간주하지 않는다.
- 실패 페이지와 원인 모듈만 재생성하고 이력을 남긴다.
- 최종 보고서는 이 디렉터리가 아니라 `../reports/YYYY/YYYY-MM-DD/<run-name>/`에 저장한다.
